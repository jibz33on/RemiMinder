Git Learning Notes:

Commands I know:
1. git add - Add files to staging
2. git commit - Save changes
3. git push - Send to remote
4. git clone - Copy repository
5. git fetch - Get updates from remote
6. git pull - Get and merge updates

Commands I'm learning:
7. git branch - Create/list branches
8. git checkout - Switch branches
9. git status - Check current state
10. git log - See commit history
